# Images Directory

This folder stores all image assets used across the PersonalWebsite pages.

## Image Naming Convention

- Use lowercase names with hyphens for separating words
- Examples: `profile-photo.jpg`, `project-thumbnail.png`, `hero-banner.jpg`

## Current Images

| Filename | Usage | Size | Format |
|----------|-------|------|--------|
| `profile-photo.jpg` | Hero section & About section profile image | - | JPG |

## Adding New Images

1. Add your image files to this folder
2. Update the HTML `src` attributes to reference the new image path
3. Use relative paths: `./images/your-image.jpg`

## Image Optimization Tips

- Compress images before adding to the repository
- Use appropriate formats (JPG for photos, PNG for graphics with transparency)
- Keep file sizes under 500KB when possible for faster loading
